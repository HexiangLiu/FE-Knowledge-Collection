# 什么是作用域链

查找变量会从当前上下文变量对象查找，没找到再从父级查找，多个执行上下文的变量对象构成的链表叫作用域链

# 执行上下文的属性

- 变量对象
- 作用域链
- this

# 作用域链创建过程

1. 函数创建，创建[[scope]]属性即父作用域链
2. 创建执行上下文，压入执行上下文栈
3. 复制函数[[scope]]属性至作用域链
4. 创建变量对象，arguments, 形参, 函数声明, 变量声明
5. 变量对象压入作用域链
6. 执行函数
7. 更新变量对象值
8. 执行完毕，退出上下文栈
